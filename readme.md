# Vienna Subway Map
## _PSConf.eu_

20 june 2022, 7 OGD engineers went to PSConf.eu in Vienna. This little webpage tells about our experiences there. It's a basic HTML-imagemap, pulling content from a json file. This webpage is hosted by Azure Static Webapp based on a Github Action.
